# Space Mission Planner Frontend

React + Three.js application for space mission visualization.

## Setup

```bash
cd 04-frontend
npm install
npm start
```

## Development

Runs on http://localhost:3000

Backend should be running on http://localhost:8000

## Features

- 3D solar system visualization
- Planet selection for trajectory planning
- Trajectory display
- Time controls

## Available Scripts

| Script | Purpose |
|---|---|
| `npm start` | Development server |
| `npm run build` | Production build |
| `npm test` | Run tests |
